# Convolutions and Pooling

## Description
This project implements the fundamental operations used in Convolutional Neural Networks (CNNs) using only NumPy. It covers image convolution, padding, strides, and pooling operations without relying on built-in convolution functions.

## Learning Objectives
By completing this project, I learned how to:

- Understand the concept of convolution in image processing.
- Apply kernels (filters) to extract image features.
- Implement same and valid padding.
- Use strides during convolution.
- Perform convolutions on multi-channel images.
- Implement max pooling and average pooling.
- Calculate the output dimensions of convolution and pooling layers.

## Requirements
- Python 3.5
- NumPy 1.15
- pycodestyle 2.5

## Files
- `0-convolve_grayscale_valid.py` – Convolution with valid padding.
- `1-convolve_grayscale_same.py` – Convolution with same padding.
- `2-convolve_grayscale_padding.py` – Convolution with custom padding.
- `3-convolve_grayscale.py` – General grayscale convolution.
- `4-convolve_channels.py` – Convolution on images with multiple channels.
- `5-convolve.py` – General convolution on multiple images and kernels.
- `6-pool.py` – Max and average pooling implementation.
